#  简单配置

### 下载项目

```
git clone https://gitee.com/chukk8/linux-c.git
cd linux-c 
chmod +x abc
```



### 下载mmdb

```
rm -rf ~/.config/XXX && mkdir -p ~/.config/XXX
cp Country.mmdb ~/.config/XXXX/

```

### 启动代理

```
./abc -f hk.yaml
```
